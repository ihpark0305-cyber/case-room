# 케이스 연습실 — 2026 신규간호사 1차(직무) 면접

처음 보는 케이스를 랜덤으로 뽑아 5분 안에 푸는 4인 스터디 연습 도구.

**열기:** 라이브 링크(아래) 또는 `index.html` 더블클릭 — 서버·인터넷 없이도 동작.

## 구성
- **케이스 룸**: 모드(아산 5+1분 / 삼성 5분 / 자유)·계통·★빈출 필터 → 랜덤 뽑기 → 시나리오·바이탈 먼저 → 타이머 → 정답 펼침(1순위 진단=생명위협 우선)
- 케이스 뱅크(95) · 11일 일정(체크리스트 저장) · 병원 8 · Q&A 룸(218, 기본간호 9개 소주제 포함) · 플래시카드(수치 24) · 채점 보드(4인 Lv1/2/3 + 약점 통계)

## 데이터/뷰 분리
- 원본: `data/*.json` (cases · schedule · hospitals · qa · reference)
- 뷰: `_template.html` → `python build.py` → `index.html` / `interview-prep.html` (데이터 inline 단일 파일)

## 주의
- 합격 후기·족보 기반 학습 자료 — **비공개 스터디 용도**. 검색 비노출(noindex) 처리됨.
- 임상 내용(진단·중재·수치)은 교재로 교차검증 권장.
- 채점·체크리스트는 각자 브라우저에 저장(localStorage), 사람 간 공유 안 됨.
